# bitcoin-price

Gets the current bitcoin price from mtgox.

## Installation

Install [Leiningen](https://github.com/technomancy/leiningen)

## Usage

```
lein run
```

or

```
lein uberjar
java -jar target/bitcoin-price-0.1.0-SNAPSHOT-standalone.jar
```
